# 🔧 VW Diagnóstico

**Sistema profissional de diagnóstico automotivo para veículos Volkswagen.**

Ferramenta completa para técnicos e mecânicos realizarem diagnósticos, consultarem códigos de falha (DTC), e gerenciarem manutenções de veículos VW.

---

## 🔗 Acesse o Sistema

### 👉 [Abrir VW Diagnóstico](https://vhu-code.github.io/VW_Diagnostico/)

---

## ✨ Funcionalidades

- 🔍 Consulta de códigos de falha (DTC) Volkswagen
- 📊 Diagnóstico completo por sistema do veículo
- 🚗 Banco de dados de modelos VW (Gol, Polo, Virtus, T-Cross, Nivus, Saveiro, Amarok e mais)
- 📋 Histórico de diagnósticos realizados
- 🔑 Sistema de licenças para acesso profissional
- 📱 Interface responsiva — funciona no celular e computador
- ⚡ Consulta rápida e offline (PWA)

## 📋 Como Usar

1. Acesse o link acima
2. Ative sua licença com a chave fornecida
3. Selecione o modelo do veículo
4. Realize o diagnóstico completo
